# project_rules.md

This is a list of rules for the current project.

## Rules

- Use `ruff` for linting and formatting.
- Use `uv` for python project management.
- Use `uv` for running the project.

## Additional considerations

- Project will be run using `docker` or `docker compose` in production.
- Project documentation is located in `docs/` and should be refered to when making design decisions.
